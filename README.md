# Bank Analytics Project

## 📌 Overview
This project analyzes loan data to identify trends, risks, and performance.

## 🛠 Tools Used
- Excel
- SQL
- Power BI
- Tableau

## 📊 KPIs
- Year-wise Loan Amount
- Conversion Rate
- Verified vs Non-Verified Payments
- State-wise Loan Status

## 📈 Insights
- Verified users have higher repayment rates
- Certain states show higher default rates

## 📂 Dataset
Available in files folder

## 🚀 Conclusion
This project helps in understanding loan performance and risk factors.
